# HireFlow AI

An AI-powered Applicant Tracking System (ATS) built using the MERN Stack.

## Tech Stack

- React
- Node.js
- Express
- MongoDB
- Tailwind CSS
- JWT Authentication
- Cloudinary
- Gemini API

## Features

- Candidate Dashboard
- Recruiter Dashboard
- Resume Upload
- AI Resume Analysis
- Job Management
- Application Tracking

> Project currently under development.
